# 🎥 Agentic AI YouTube Video Analyzer

An Agentic AI application built using **Python, Streamlit, Agno, and Groq** that analyzes YouTube videos by extracting their content and generating AI-powered summaries and responses. The application leverages an AI agent with YouTube tools to understand video content and provide meaningful insights through an interactive interface.

---

## 🚀 Features

- 📺 Analyze YouTube videos using their URL
- 🤖 Agentic AI-powered video understanding
- 📝 Generate concise video summaries
- ❓ Ask questions about video content
- ⚡ Fast responses using Groq Llama models
- 🌐 Interactive Streamlit interface

---

## 🔄 Workflow

1. User enters a YouTube video URL.
2. The AI Agent receives the request.
3. Agno's YouTubeTools extract the video content.
4. The Groq Llama model analyzes the extracted information.
5. The Agent generates a summary or answers user questions.
6. The response is displayed in the Streamlit application.
